## О проекте

Консольное приложение на Java для анализа текста.
Поддерживает подсчёт символов, слов, частотный анализ, поиск и редактирование текста.

### Возможности

* Подсчёт:

    * символов;
    * слов;
    * цифр;
    * букв;
    * специальных символов;
* Анализ регистра;
* Частотный анализ слов и символов;
* Поиск слов и позиций;
* Замена, удаление и вставка текста (`StringBuilder`);
* Сортировка слов:

    * по алфавиту;
    * по длине;
    * по частоте.

### Технологии

* Java 17+
* String / StringBuilder
* ArrayList / HashMap

### Структура проекта

* `Main` — запуск программы
* `TextAnalyzer` — анализ текста
* `TextEditor` — изменение текста
